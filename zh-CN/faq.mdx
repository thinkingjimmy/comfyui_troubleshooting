你可以去到我们的 Github 页面里的 [Issue](https://github.com/thinkingjimmy/comfyui_troubleshooting/issues) 专区提交你的问题，你的问题有可能被我回答，或被其他社群的朋友回答。

另外，提交问题的时候，需要注意：

1. **请将 ComfyUI 的报错关键词作为标题。**

注意不要将整段报错都粘贴成标题，比如下面这个报错，非常长，但关键词其实很短，是 Value not in list：

<Frame>
  <img src="/images/intro/001.png" />
</Frame>

2. **尽量详细描述报错的问题。**

Issue 的详情里，最好将上述的报错完整截图，放到提问里。并且详细描述下，发生这个报错的时候，你做了什么操作。比如将某个参数调整了 x。最后，如果可以的话，请将你的电脑相关配置与 ComfyUI 版本信息也一起提交，如果你用的是别人的工作流，那麻烦将工作流文件也一并附上。这样能更好地解决你的问题。

这个是一个写得比较好的 [Issue 模板](https://github.com/thinkingjimmy/comfyui_troubleshooting/issues/2)，你可以参考一下。
